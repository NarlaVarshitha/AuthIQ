# AuthIQ Design System

- Primary color: #2563eb (blue)
- Secondary color: #1d4ed8
- Background: #f8fafc
- Text: #1e293b (dark), #475569 (medium)
- Success: #10b981
- Warning: #f59e0b
- Danger: #ef4444

- Font: system‑sans (Inter if available), fall back to Arial, sans‑serif
- Spacing: use consistent spacing (e.g., 8px, 16px, 24px, 32px up)
- Layout: use flex/grid, responsive mobile‑first
- Components:
  - Card: light border, rounded edges
  - Button: primary (blue), secondary (outline)
  - Badge: small pill‑shaped tag
